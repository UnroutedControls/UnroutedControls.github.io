❓ How to add a post to brainstew.html
✕

After hitting GENERATE PASTE CODE, copy the output and open brainstew.html in a text editor.


  1. Find the comment:
    <!-- ADD POSTS HERE -->
    Paste the inbox row HTML there.

  2. Find the comment:
    <!-- add more markdown posts here -->
    Paste the markdown script block there.

  3. Update the statusbar message count if you want.

  4. Save and upload to Neocities. Done!! ♥

Or just upload a .md file directly to brainstew.html using the Open .md button in the toolbar — it'll render immediately without editing any HTML!
